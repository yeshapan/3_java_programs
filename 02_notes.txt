Java Text Output Methods

  1. Text Output

        i) println: adds new line at end of output
            example:
                    System.out.println("Hello World!");
                    System.out.println("I am learning Java.");
                    System.out.println("It is awesome!");

        ii) print: does not add new line at end of output
            example:
                    System.out.print("Hello World! ");
                    System.out.print("I will print on the same line.");

NOTE: try to prefer using println for clarity in output

  2. Numbers Output
        we can also use println to output numbers
        examples:
                System.out.println(3);
                System.out.println(358);
                System.out.println(3 + 3);
                System.out.println(2 * 5);
